I replaced the createNewPerson() function with a constructor function. Here are the two functions to compare.

normal function

function createNewPerson(name) { const obj = {}; obj.name = name; obj.greeting = function() { alert('Hi! I\'m ' + obj.name + '.'); }; return obj; }

constructor function

function Person(name) { this.name = name; this.greeting = function() { alert('Hi! I\'m ' + this.name + '.'); }; }

Note, I didn't assign empty brackets in the constructor function. It also uses the 'this' keyword instead of the object name. 'this' is passed from the object's 'name' property. It's important to use the 'this' keyword because the value of 'name' will change when objects are created.
