Objects

Objects have properties and methods. Each property is a value. It is written as name:value. The value can be a string, in which case it must be in "".

Methods are actions that can be performed on objects. They are written name:function(){}

Object Loop

Use for...in to loop through the properties of an object. The below example will display all the object properties.

let txt = "";
let person = {fname:"john", lname:"Doe", age:25};
for (x in person) {
txt += person.x;
}
document.getElementById("output").innerHTML = txt;

I can add another property to the above object by doing this:
person.nationality = "English";

I can also delete a property by doing this:
delete person.age;

Methods

Methods allow me to use objects by putting them in a function. I can add a function directly into the value of a property. Here's an example.

let person = {
fname:"john",
lname:"Doe",
age:25,
fullName() {
return this.firstName + " " + this.lastName;
}
};
name = person.fullName();
document.getElementById("output").innerHTML = name;

The this word refers to the object name.

I don't full understand the this keyword. I just use the property name and it works the same. What is the benefit of using this? (Answer, it can be used in a function to call multiple object names. More details in the This section.

I can use the 'get' word in place of a function to avoid either reassigning a function or putting the () to call the object property. In the above example I would do this instead.

let person = {
fname:"john",
lname:"Doe",
age:25,
get fullName() {
return this.firstName + " " + this.lastName;
}
};
document.getElementById("output").innerHTML = person.fullName;

Using Objects

convert an object to an array by using Object.values(name);

let person = {fname:"john", lname:"Doe", age:25};
let array = Object.values(person);

This turns the object properties into an array.

This

The 'this' keyword calls the name of an object. This is useful if there is more than one object.

let user = { name: "John" };

let admin = { name: "Admin" };
function display() {
alert( this.name);
}

Now I can call the function by assigning the function as a value to the object name

user.f = display;
admin.f = display;

And this runs the function

user.f();
admin.f();
