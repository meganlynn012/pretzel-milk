Event Listeners

These are similar to calling a function in the html with onclick. But this acts in the js file. First, call the element in the html with an id or query selector. Add .addEventListner after, then an => function. Like so.

document.querySelector(".listContainer").addEventListener((item) event => {
some code here
call a function
function()
});

Very useful.

Submit inputs

Instead of a button, use an input with type submit. Add a value as an attribute to change the text.

<input type="submit" value="Button Text"></input>

wrap that in a form. Then do the event listener as stated above and grab the form with the query selector. This will allow the user to either click the submit button or simply press enter.

Functions +

Iffy functions are cool. IIFE, or immediately invoved function expression is a function that will load when the page loads. Like an event listener set to activate on load, but without all the code to state that. Put the whole function in () like this. Just check the console to see that in action.

(function(){
const temp = 'World';
console.log(`Hello ${temp}`);
})();

This function type will only run once. It's great for displaying a message when the page loads.
